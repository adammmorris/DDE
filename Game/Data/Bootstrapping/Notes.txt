HOW TO RUN BOOTSTRAPPING

(1) Create "Bootstrapping" folder within folder containing relevant ".RData" file
(2) Create "model" (or model_all, etc.) folder within the bootstrapping folder
(3) Set Bootstrap.sh accordingly
(4) Use this q-sub command: qsub -t 1-250 -o /dev/null -e /dev/null Bootstrap.sh
(5) Set ParseBootstrap.R correctly, and paste it into an R shell
(6) Record here!
